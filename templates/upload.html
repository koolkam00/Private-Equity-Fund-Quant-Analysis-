Deal-Level File

Download Current Deal Template

The download includes generated example data so users can see exactly how to populate each field.

Required core fields: Firm Name, Company Name, Fund, Sector, Status, Investment Date, Equity Invested, Realized Value, Unrealized Value, Entry/Exit Revenue, EBITDA, TEV, Net Debt.

Optional firm-level field: Firm Currency (ISO-3 code such as USD/EUR/GBP). If omitted or blank, default currency is USD.

Additional qualitative fields supported for filtering: Geography, Year Invested, Ownership %, Exit Type, Lead Partner, Security Type, Deal Type, Entry Channel. Optional track-record fields: Fund Size, Net IRR, Net MOIC, DPI. Missing optional fields are auto-filled with default labels.

Optional multi-sheet support is available for advanced analysis pages: Cashflows, Deal Quarterly, Fund Quarterly, and Underwrite. The Deals sheet remains the only required sheet.

Upload behavior: each workbook must contain exactly one Firm Name in the Deals sheet. If the firm does not exist yet, it is created automatically. If Firm Currency is provided, it sets that firm's base currency (ISO-3); fund replacement runs within that firm only.
